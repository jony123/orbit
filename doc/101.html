Multiplayer Network Support

  • Up to 16 players supported in a client/server model.
  • Clients and servers can be any mix of supported platforms.
  • Each player chooses his or her name and spaceship model.
  • Chat with other players using text messages.
  • Server control of gravity, full stop, planetary orbits, etc.

Direct Support for AC3D models

Object models in AC3D format can be loaded directly. Smooth shading, texture mapping, and material properties are supported.

Planetary Orbits

  • Orbital paths of planets and moons can be displayed.
  • Orbital motion can be enabled.
  • Variable time compression.

Other Changes

  • Linux ESD sound support (recompile with ESD defined).
  • Full Stop can be toggled on or off (with the fullstop preferences variable).
  • Adjustable Field Of View (< and >, keys).
  • Warp engines can be set to exponential acceleration (superwarp preferences variable).
  • View direction can be locked on a planet or target (v key).
  • Range to nearby objects shown next to object name.
  • Ranges shown to planet surface instead of planet center.
  • Aiming cursor only shown when target in range of current weapon.
  • Missions can be loaded by name (ctrl-L key).
  • Adjustable near and far clipping planes ({, }, (, ) keys).
  • Player name can be changed in the game (ctrl-N key).
  • Throttle changes independent of frame rate.
  • Some minor bug fixes.
